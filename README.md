# VitalAI – AI Health Monitoring System

## Overview
VitalAI is an AI-powered health monitoring system that analyzes user lifestyle data such as sleep, stress, screen time, physical activity, and symptoms to generate a health score, summary, doctor advice, and personalized recommendations.

The system also sends the health report directly to the user's email.

---

## Features
- User authentication (Signup / Login)
- AI health analysis
- Health score calculation
- Doctor advice generation
- Lifestyle recommendations
- Health tips
- Email health report delivery
- Interactive dashboard with health score visualization

## Installation

1. Clone the repository

2. Navigate to project folder

cd vitalai-health-monitoring

3. Create virtual environment

python -m venv venv

4. Activate environment

Windows:
venv\Scripts\activate


5. Install dependencies
pip install -r requirements.txt


6. Run the application
python app.py
